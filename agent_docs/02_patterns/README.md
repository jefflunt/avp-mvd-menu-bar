# 02 Patterns

This directory holds project implementation patterns and programming guidelines.

Currently, we adopt:
- **Model-View-ViewModel (MVVM)** in SwiftUI.
- **Swift 6 Concurrency**: Non-isolated static utilities to prevent actor crossing violations.
- **Monochrome Status Bar Items**: Standard macOS system styling.
- **Direct System APIs**: No shell processes to support App Store sandboxing.
